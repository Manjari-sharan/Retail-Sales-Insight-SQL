# Retail-Sales-Insight-SQL
This project focuses on analyzing retail transaction data to uncover customer purchasing behavior, sales trends, and category performance. 
Using SQL-driven analysis, the project transforms raw transactional data into actionable business insights.

# Objective
- Analyze retail sales data to identify key revenue drivers
- Understand customer demographics and purchasing patterns
- Evaluate category-level performance
- Generate insights to support data-driven business decisions

# Key SQL Concepts Used
- Aggregations (SUM, COUNT, AVG)
- Grouping (GROUP BY)
- Filtering (WHERE, HAVING)
- Sorting (ORDER BY)
- Window functions, CTEs
- Date-based analysis
- Data exploration & cleaning techniques

# Tools & Technologies
- SQL
- Jupyter Notebook
- Python (for execution environment)

# Analysis Performed
1. Data Exploration & Cleaning
- Validated total record count
- Checked for missing/null values
- Identified unique customers and transactions
- Verified date range and category distribution

2. Sales Analysis
💰 Total Sales by Gender
- Compared revenue contribution across customer segments
- Identified dominant purchasing group
🛍️ Sales by Product Category
- Evaluated category-wise revenue performance
- Highlighted top-performing product segments
📅 Sales Trend Analysis
- Analyzed top sales days
- Identified peak transaction periods
👥 Sales by Age Group
- Examined spending behavior across age demographics
- Derived insights on high-value customer segments

# Key Findings
- Product strategies : Certain product categories contribute disproportionately to total revenue, indicating high-performing segments for business focus
- Customer Demographics: Customer demographics (age & gender) show distinct purchasing patterns, enabling targeted marketing strategies
- High-Value Orders: Several transactions exceed 1000 in total sales, indicating premium purchases.
- Sales Trends: Sales trends reveal peak transaction periods, useful for inventory planning and promotions
- Customer Insights: A subset of customers drives a significant portion of sales, highlighting opportunities for customer retention strategies

# Reports Generated
- Sales Summary Report — Category-wise sales and order volume
- Trend Analysis — Monthly and shift-based sales insights
- Customer Insights — Top customers and unique customer counts
